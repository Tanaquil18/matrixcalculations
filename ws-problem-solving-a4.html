    Objectives

    • Be able to use linear systems and reduced row echelon form to solve problems in different contexts where the linear system is not immediately apparent.
    • Be able to identify unknowns and constraints, or in other words, variables and equations involving those variables.
    • Build on our past work interpreting solutions to a linear system from the reduced row echelon form by further restricting to solutions which make sense in the context of a given problem.

    1.

    Suppose we know that in a football game there were \(24\) points scored from \(8\) scoring occasions and also that the number of successful extra point kicks was equal to the number of successful two point conversions. Find all ways in which the points may have been scored in this game.
    Answer.
    The only way which meets all the conditions and makes sense is by scoring \(8\) field goals.
    Solution.
    The augmented matrix from this system is \(\left[\begin{array}{ccccc} 1\amp 1\amp 1\amp 1\amp 8\\6\amp 1\amp 2\amp 3\amp 24\\0\amp 1\amp -1\amp 0\amp 0\\ \end{array} \right]\text{.}\) From this we find the solution
    \begin{align*} t\amp =\frac83-\frac13f\\ x\amp =\frac83-\frac13f\\ c\amp =\frac83-\frac13f\text{.} \end{align*}
    The only time each of these variables are nonnegative integers is when \(f=2\) or \(f=8\text{.}\) If \(f=2\text{,}\) then we have \(2\) touchdowns, \(2\) extra points and \(2\) two point conversions (and \(2\) field goals); this doesn’t make sense since the extra points and two point conversions follow touchdowns. If \(f=8\text{,}\) then we have no touchdowns, extra points or two point conversions (just \(8\) field goals). This is the only solution; all points were scored from field goals.

    2.

    Pure elemental phosphorous and hypochlorous acid \(HClO\) react in water to make phosphoric acid \(H_3PO_4\) and hydrochloric acid \(HCl\text{.}\) Balance the chemical equation
    \begin{equation*} \fillinmath{X} P_4 + \fillinmath{X} HClO+\fillinmath{X} H_2 O \longrightarrow \fillinmath{X} H_3PO_4+\fillinmath{X} HCl\text{.} \end{equation*}
    Answer.
    The balanced equation is
    \begin{equation*} P_4 + 10 HClO+ 6 H_2 O \longrightarrow 4 H_3PO_4+ 10 HCl\text{.} \end{equation*}
    Solution.
    We are looking for
    \begin{equation*} x_1 \, P_4 + x_2 \, HClO+ x_3\, H_2 O \longrightarrow x_4 \, H_3PO_4+ x_5 \, HCl\text{.} \end{equation*}
    Consider the number of atoms of each element before and after the reaction.
    \begin{align*} P\amp: \amp 4x_1 \amp=x_4 \\ H\amp: \amp x_2 + 2x_3 \amp = 3x_4+x_5 \\ Cl\amp: \amp x_2 \amp=x_5\\ O\amp: \amp x_2+x_3 \amp =4x_4 \end{align*}
    Turning this system into an augmented matrix and row reducing results in
    \begin{equation*} \begin{bmatrix} 4 \amp 0 \amp 0 \amp -1 \amp 0 \amp 0 \\ 0 \amp 1 \amp 2 \amp -3 \amp -1 \amp 0 \\ 0\amp 1\amp 0 \amp 0 \amp -1 \amp 0 \\ 0\amp 1 \amp 1 \amp -4 \amp 0 \amp 0 \end{bmatrix} \rrefarrow \begin{bmatrix} 1 \amp 0 \amp 0 \amp 0 \amp \frac{-1}{10} \amp 0 \\ 0 \amp 1 \amp 0 \amp 0 \amp -1 \amp 0 \\ 1 \amp 0 \amp 0 \amp 0 \amp \frac{-3}{5} \amp 0 \\ 1 \amp 0 \amp 0 \amp 0 \amp \frac{-2}{5} \amp 0 \end{bmatrix} \end{equation*}
    We want the smallest possible numbers such that no value is negative or fractional, so let \(x_5=10\text{.}\) Then
    \begin{align*} x_1 \amp=1 \\ x_2 \amp=10 \\ x_3 \amp=6 \\ x_4 \amp=4 \\ x_5 \amp=10 \end{align*}
    and the balanced equation is
    \begin{equation*} P_4 + 10 HClO+ 6 H_2 O \longrightarrow 4 H_3PO_4+ 10 HCl\text{.} \end{equation*}

    3.

    You might know that two points determine a line. That’s because in a line \(y=mx+b\) there are two unknowns: the coefficient of the \(x\) term and the constant term. Note that in Reading Question 1 we needed three points to determine a specific quadratic function because there were three unknowns: the coefficient of the \(x^2\) term, the coefficient of the \(x\) term and the coefficient of the constant term.
    (a)
    Find the smallest degree polynomial which goes through the points \((-2,15)\text{,}\) \((-1,4)\text{,}\) \((1,0)\) and \((2,-5)\text{.}\)
    Answer.
    \(f(x) = -x^3+x^2-x+1\)
    (b)
    Go through the same steps to attempt to find a quadratic function through the points \((0,0)\text{,}\) \((1,1)\text{,}\) and \((2,2)\text{.}\) What do you get, and why does the answer make sense?
    Answer.
    The coefficient of the \(x^2\) term is \(0\text{,}\) giving the linear function \(y=x\text{.}\) The three given points all lie on a line.

    4.

    In a basketball game, where points are scored either by a \(3\) point shot, a \(2\) point shot or a \(1\) point free throw, there were \(80\) points scored from \(30\) successful shots. Find all ways in which the points may have been scored in this game.
    Answer.
    There are \(6\) possible solutions:
    • \(0\) free throws, \(10\) two-point shots, \(20\) three-point shots
    • \(1\) free throws, \(8\) two-point shots, \(21\) three-point shots
    • \(2\) free throws, \(6\) two-point shots, \(22\) three-point shots
    • \(3\) free throws, \(4\) two-point shots, \(23\) three-point shots
    • \(4\) free throws, \(2\) two-point shots, \(24\) three-point shots
    • \(5\) free throws, \(0\) two-point shots, \(25\) three-point shots
    Solution.
    Let \(x_1\text{,}\) \(x_2\) and \(x_3\) represent the number of free throws, 2 point and 3 point shots taken. The augmented matrix from this system is \(\left[\begin{array}{cccc} 1\amp 1\amp 1\amp 30\\1\amp 2\amp 3\amp 80 \end{array} \right]\text{.}\) From this we find the solution
    \begin{align*} x_1\amp =-20+x_3\\ x_2\amp =50-2x_3\text{.} \end{align*}
    In order for \(x_1\) and \(x_2\) to be nonnegative, we need \(20\leq x_3\leq 25\text{.}\) Thus there are \(6\) different scenarios: the “first” is where \(20\) three point shots are taken, no free throws, and \(10\) two point shots; the “last” is where \(25\) three point shots are taken, \(5\) free throws, and no two point shots.
