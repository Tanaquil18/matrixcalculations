Definition 2.3.2. Invertible Matrices and the Inverse of A.

Let \(A\) be an \(n\times n\) matrix. If there exists an \(n\times n\) matrix \(C\) such that \(A C = I = C A\text{,}\) then
  1. \(A\) is invertible.
